INTRODUCTION
------------

The mark visited nodes module stores the nids of nodes and the uids of the users that viewed the nodes in a table in the database.


INSTALLATION
------------

1. With drush: drush dl mark_visited_nodes / drush en mark_visited_nodes
2. Download the module on drupal.org/project/mark_visited_nodes. Paste the folder in your module folder (sites/all/modules/). Go to yourdrupalproject.com/admin/modules select the Mark visited nodes module and  press "save configuration".


REQUIREMENTS
------------

1. The views module (https://www.drupal.org/project/views), needs to be installed and enabled.
